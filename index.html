📋 TiviMate 订阅步骤

1. 打开 TiviMate → 设置 → 播放列表
2. 添加播放列表 → 输入 URL
3. 粘贴上方 jsDelivr CDN 链接
4. 或使用手机扫描右上角二维码
5. 等待加载完成即可观看 🎉

📡 EPG 电子节目单

在播放器 EPG 设置中添加以下源：

EPG.PW https://epg.pw/xmltv/china.xml

fanmingming https://raw.githubusercontent.com/fanmingming/live/main/e.xml

推荐使用 fanmingming EPG，更新更及时

⚡ GitHub Pages 托管

1. 登录 GitHub → 进入仓库 Settings
2. 左侧 Pages → Source 选择 main 分支
3. 保存后访问 https://用户名.github.io/iptv-engine
4. 即可全球极速访问本 Dashboard 🌍
